# Demo_Web_Ecommerce
### 1. npm run server-install
### 2. npm run client-install
### 3. connect to mongodb and add info in .env
### 4. add your paypal client id in client/src/components/mainpages/cart/PaypalButton.js
### 5. npm run dev
